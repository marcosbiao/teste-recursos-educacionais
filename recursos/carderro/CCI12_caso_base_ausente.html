Caso base ausente

Etapa 1 · Observe

A chamada recursiva reduz o argumento, mas nunca encontra uma condição de encerramento

Problema

Implemente uma função recursiva que calcule o fatorial de um número inteiro não negativo. Por definição, 0! e 1! valem 1.

Solução em análise

A função sempre executa uma nova chamada. Mesmo quando n chega a 1 ou 0, nenhuma condição devolve um resultado diretamente.

Onde está o erro?

Análise do diagnóstico

Uma recursão precisa de um caminho que não gere nova chamada

Diagnóstico

A alternativa A descreve o erro. A função possui passo recursivo, mas não possui caso base.

Causa conceitual

O caso base define uma entrada simples cujo resultado pode ser devolvido imediatamente. Sem ele, toda chamada depende de outra chamada.

Quando aparece

Quando o estudante pensa apenas em como reduzir o problema, mas não define quando a redução deve parar.

Como a pilha cresce

Cada chamada fica aguardando o retorno da próxima.

Como corrigir?

Etapa 3 · Corrija

Defina o caso base antes de realizar a chamada recursiva

Correção da função fatorial
−int fatorial(int n) {
− return n * fatorial(n - 1);
−}
+int fatorial(int n) {
+ if (n <= 1) return 1;
+ return n * fatorial(n - 1);
+}
Por que funciona?

Quando n <= 1, a função devolve 1 sem criar nova chamada. As chamadas anteriores então recebem os resultados e podem retornar em ordem inversa.

  • Definir qual entrada possui resultado conhecido diretamente.
  • Testar o caso base antes da chamada recursiva.
  • Garantir que o passo recursivo se aproxima do caso base.
  • Confirmar que o caso base não cria nova chamada.
  • Simular chamadas e retornos com um valor pequeno.

Síntese diagnóstica

Uma função recursiva só termina quando existe uma entrada que produz resposta direta, sem nova chamada.

Erro

Definir apenas a chamada recursiva e omitir o caso base.

Sinal

A pilha de chamadas cresce e nenhum resultado retorna.

Diagnóstico

Localizar a condição que deveria devolver um valor diretamente.

Correção

Testar o caso base antes de executar o passo recursivo.
