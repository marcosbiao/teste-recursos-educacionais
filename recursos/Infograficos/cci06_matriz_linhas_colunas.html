Ideia central

Uma matriz combina duas dimensões

Cada elemento ocupa o encontro entre uma linha e uma coluna. Em C, o acesso usa dois índices: o primeiro localiza a linha e o segundo localiza a coluna.

matriz[linha][coluna]

Como os laços percorrem a matriz

1. Linha atual

Inicie o laço externo

O índice i escolhe qual linha será percorrida.

i = 0
2. Coluna atual

Inicie o laço interno

Para cada linha, o índice j começa novamente na primeira coluna.

j = 0
3. Célula atual

Acesse o elemento

O par de índices identifica um único valor.

matriz[i][j]
4. Avance

Complete a linha

Quando j termina, i avança e uma nova linha começa.

j++ → i++

Exemplos

Nível básico

Exibir uma matriz 2 × 3

Visite cada célula e mostre os valores por linha.

ParteAplicação
Linhas2, índices 0 e 1
Colunas3, índices 0, 1 e 2
Acessom[i][j]
Saída4 7 2 / 8 5 9
exibir_matriz.c
#include <stdio.h>

int main() {
    int m[2][3] = {{4, 7, 2}, {8, 5, 9}};
    int i, j;

    for (i = 0; i < 2; i++) {
        for (j = 0; j < 3; j++) {
            printf("%d ", m[i][j]);
        }
        printf("\n");
    }

    return 0;
}

#include <stdio.h>

int main() {
    int m[2][3] = {{4, 7, 2}, {8, 5, 9}};
    int i, j, soma = 0;

    for (i = 0; i < 2; i++) {
        for (j = 0; j < 3; j++) {
            soma = soma + m[i][j];
        }
    }

    printf("Soma: %d\n", soma);
    return 0;
}

Maior valor e sua coordenada

Além do maior valor, registre os índices em que ele foi encontrado.

maior_na_matriz.c
#include <stdio.h>

int main() {
    int m[2][3] = {{4, 7, 2}, {8, 5, 9}};
    int maior = m[0][0];
    int linhaMaior = 0, colunaMaior = 0;
    int i, j;

    for (i = 0; i < 2; i++) {
        for (j = 0; j < 3; j++) {
            if (m[i][j] > maior) {
                maior = m[i][j];
                linhaMaior = i;
                colunaMaior = j;
            }
        }
    }

    printf("Maior: %d em [%d][%d]\n",
           maior, linhaMaior, colunaMaior);
    return 0;
}

Erros comuns

Trocar linha por coluna

Por que acontece: o aluno interpreta os índices em ordem inversa.

Como evitar: leia sempre [linha][coluna].

Usar o mesmo limite

Por que acontece: o número de linhas é usado também no laço das colunas.

Como evitar: cada dimensão deve ter seu próprio limite.

Usar apenas um laço

Por que acontece: a matriz é tratada como se fosse um vetor.

Como evitar: use um laço para linhas e outro para colunas.

Acessar fora da matriz

Por que acontece: o laço usa <= no lugar de <.

Como evitar: para 3 linhas, use índices 0, 1 e 2.

Perguntas rápidas

Em dados[2][1], qual índice representa a coluna?

O segundo índice. Nesse acesso, a linha tem índice 2 e a coluna tem índice 1.

Por que o laço interno reinicia quando uma nova linha começa?

Porque cada linha precisa ser percorrida desde a primeira coluna até a última.

Quais são os índices válidos de uma matriz declarada como int m[3][4]?

As linhas usam índices de 0 a 2 e as colunas usam índices de 0 a 3.

Fechamento pedagógico

Depois de estudar este infográfico, o aluno deve conseguir interpretar uma matriz como uma coleção bidimensional, acessar uma célula por linha e coluna e construir laços aninhados que percorram todos os elementos sem ultrapassar seus limites.

Coleção bidimensional

A matriz

Uma matriz armazena valores do mesmo tipo em uma estrutura com linhas e colunas.

Mesmo exemplo: int notas[3][4] reserva 3 linhas e 4 colunas, totalizando 12 elementos.
  • Primeira dimensão: quantidade de linhas.
  • Segunda dimensão: quantidade de colunas.
  • Cada célula possui uma coordenada própria.
×
Primeiro índice

A linha

O primeiro índice escolhe uma linha horizontal da matriz.

Mesmo exemplo: em notas[1][2], o índice de linha é 1, que corresponde à segunda linha.
  • Com 3 linhas, os índices válidos são 0, 1 e 2.
  • O laço externo normalmente controla esse índice.
  • Quando a linha muda, o percurso das colunas começa novamente.
×
Segundo índice

A coluna

O segundo índice escolhe uma coluna vertical dentro da linha atual.

Mesmo exemplo: em notas[1][2], o índice de coluna é 2, que corresponde à terceira coluna.
  • Com 4 colunas, os índices válidos são 0, 1, 2 e 3.
  • O laço interno normalmente controla esse índice.
  • O limite das colunas pode ser diferente do limite das linhas.
×
Percurso completo

Os laços aninhados

Um laço fica dentro do outro para formar todas as combinações entre linhas e colunas.

Mesmo exemplo: for (i = 0; i < 3; i++) controla as linhas e for (j = 0; j < 4; j++) controla as colunas.
  • Para cada valor de i, j percorre todas as colunas.
  • O acesso ocorre por notas[i][j].
  • Ao final de uma linha, o laço externo avança para a próxima.
