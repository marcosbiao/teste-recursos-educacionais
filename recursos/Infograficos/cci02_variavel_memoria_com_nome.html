Ideia central

O programa precisa lembrar informações

Uma variável associa um nome a um espaço de memória. Esse espaço guarda um valor compatível com seu tipo. Quando ocorre uma atribuição, o conteúdo armazenado pode ser substituído. Uma constante também possui nome, mas representa um valor que não deve mudar durante a execução.

3

Atualizar

Cada atribuição pode produzir um novo estado para o programa.

O nome permite recuperar e modificar a informação sem depender de um endereço de memória.

Constante: um nome para um valor fixo

A constante torna explícito que aquele valor tem significado e não deve mudar.

Por que usar

1

Evita repetir um número sem explicar o que ele representa.

2

Centraliza um valor que participa de vários cálculos.

3

Impede alterações acidentais depois da declaração.

No mesmo exemplo

const int BONUS = 10;

O nome BONUS comunica a função do valor 10. O programa pode usá-lo, mas não deve atribuir outro valor a essa constante.

#include <stdio.h>

int main() {
    const float TAXA_DESCONTO = 0.10f;
    float preco, desconto, precoFinal;

    scanf("%f", &preco);

    desconto = preco * TAXA_DESCONTO;

Erros comuns

Usar antes de atribuir

Por que acontece: a variável foi declarada, mas ainda não recebeu um valor válido.

Como evitar: inicialize a variável antes de utilizá-la em cálculo ou saída.

Ler a atribuição como igualdade

Por que acontece: o símbolo = é interpretado como uma afirmação matemática.

Como evitar: leia como “recebe o resultado de”.

Tentar alterar uma constante

Por que acontece: a diferença entre variável e constante não foi considerada.

Como evitar: use constante apenas para valores que devem permanecer fixos.

Repetir números sem significado

Por que acontece: o mesmo valor fixo aparece várias vezes no código.

Como evitar: atribua um nome semântico ao valor por meio de uma constante.

Perguntas rápidas

Depois de x = 5; e x = x + 2;, qual valor fica armazenado em x?

O valor armazenado passa a ser 7. O programa usa o valor atual 5, calcula 5 + 2 e grava 7 no mesmo espaço.

Uma constante pode receber outro valor depois de declarada?

Não. A constante representa um valor fixo durante a execução e não deve aparecer no lado esquerdo de uma nova atribuição.

Em idade = 20;, o símbolo = compara dois valores?

Não. Nesse contexto, ele realiza uma atribuição. O valor 20 é gravado na variável idade.
