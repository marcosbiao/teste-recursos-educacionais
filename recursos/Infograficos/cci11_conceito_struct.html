Ideia central

Uma struct cria um tipo composto

Quando várias informações descrevem a mesma coisa, elas podem ser agrupadas em uma estrutura. Assim, nome, matrícula e média deixam de ser dados soltos e passam a formar um Aluno.

Pense na struct como o molde e em cada variável desse tipo como uma entidade concreta.

Exemplos em progressão

Nível básico

Representar um aluno

Uma variável reúne matrícula e média da mesma pessoa.

ElementoPapel
struct AlunoMolde da entidade
a1Instância concreta
a1.mediaAcesso a um campo
exemplo_aluno.c
#include <stdio.h>

struct Aluno {
    int matricula;
    float media;
};

int main() {
    struct Aluno a1;

    a1.matricula = 202601;
    a1.media = 8.4;

    printf("Matricula: %d\n", a1.matricula);
    printf("Media: %.1f\n", a1.media);

    return 0;
}
Nível intermediário

Calcular o valor do estoque

Os campos preço e quantidade pertencem ao mesmo produto.

CampoValor
codigo15
preco12,50
quantidade8
ResultadoR$ 100,00
produto.c
#include <stdio.h>

struct Produto {
    int codigo;
    float preco;
    int quantidade;
};

int main() {
    struct Produto p = {15, 12.50, 8};
    float total;

    total = p.preco * p.quantidade;

    printf("Valor em estoque: R$ %.2f\n", total);
    return 0;
}
Nível intermediário

Percorrer vários livros

Um vetor armazena várias instâncias do mesmo tipo composto.

EstruturaSignificado
livros[3]Três entidades Livro
livros[i]Livro da posição atual
livros[i].disponivelCampo consultado no laço
livros.c
#include <stdio.h>

struct Livro {
    int codigo;
    int disponivel;
};

int main() {
    struct Livro livros[3] = {
        {101, 1}, {102, 0}, {103, 1}
    };

    for (int i = 0; i < 3; i++) {
        if (livros[i].disponivel == 1) {
            printf("Livro %d disponivel\n",
                   livros[i].codigo);
        }
    }

    return 0;
}

Erros comuns

Confundir molde com variável

Por que acontece: o aluno pensa que definir struct Aluno já criou um aluno.

Como evitar: depois do molde, declare uma instância como struct Aluno a1;.

Esquecer o operador ponto

Por que acontece: tenta-se usar media sem indicar a qual entidade o campo pertence.

Como evitar: escreva sempre a instância e o campo, como a1.media.

Agrupar dados sem relação

Por que acontece: a struct vira apenas um recipiente genérico.

Como evitar: inclua apenas campos que descrevem uma mesma entidade ou conceito.

Perguntas rápidas

Uma struct substitui os tipos básicos?

Não. Ela combina campos que continuam sendo int, float, char ou outros tipos.

Qual é a diferença entre struct Aluno e aluno1?

struct Aluno é o molde do tipo composto. aluno1 é uma variável concreta criada a partir desse molde.

Como acessar a média armazenada em aluno1?

Use o operador ponto: aluno1.media.

Entidade

O elemento do problema que precisa ser descrito.

No exemplo, o programa precisa representar um aluno. Nome, matrícula e média não são entidades separadas. São características da mesma entidade.

Pergunta útil: quais informações precisam permanecer juntas para representar corretamente um aluno?

Campos

As partes que compõem a entidade.

Cada campo recebe um nome que indica seu papel dentro da estrutura. No tipo Aluno, os campos podem ser nome, matricula e media.

aluno1.media significa: selecione o campo media da entidade armazenada em aluno1.
×

Tipos dos campos

Uma struct pode reunir tipos diferentes.

O campo nome pode ser um vetor de caracteres, matricula pode ser inteiro e media pode ser real. A struct não apaga essas diferenças. Ela apenas organiza os campos sob uma mesma entidade.

  • char nome[40]: texto
  • int matricula: número inteiro
  • float media: número real
×

Instância

Uma variável concreta criada a partir do molde.

A definição da struct descreve como um aluno será organizado. A declaração abaixo cria um aluno concreto na memória:

struct Aluno aluno1;

Também é possível criar aluno2, aluno3 ou um vetor de alunos. Todos seguem o mesmo molde, mas guardam valores próprios.

×

Acesso com ponto

O ponto liga uma instância a um de seus campos.

Para atribuir ou consultar um dado, indique primeiro a variável da struct e depois o campo desejado.

aluno1.media = 8.4;
printf("%.1f", aluno1.media);

O campo media existe dentro da variável aluno1. Outro aluno terá seu próprio campo media.
