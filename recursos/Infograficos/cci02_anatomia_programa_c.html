Ideia central

Um programa não é um bloco único

Em C, cada parte tem uma função. A biblioteca disponibiliza recursos, a função main marca o início da execução, as variáveis guardam dados, os comandos realizam ações e o retorno encerra a função principal.

Clique em uma parte
programa.c 
#include <stdio.h>
 
int main() {
    float nota1, nota2, media;
 
    scanf("%f", &nota1);
    scanf("%f", &nota2);
    media = (nota1 + nota2) / 2.0;
    printf("Media: %.2f\n", media);
 
    return 0;
}
Parte essencial do programa

Biblioteca

Disponibiliza declarações de funções que o programa pretende utilizar.

Neste exemplo: stdio.h permite usar scanf e printf.

A diretiva começa com #include e aparece antes da função main.

#include <stdio.h>

Função main

É o ponto a partir do qual a execução do programa começa.

Observe: o tipo int informa que a função retorna um valor inteiro.

As chaves delimitam o bloco em que ficam variáveis, comandos e retorno.

int main() { // instruções }

Declaração de variáveis

Define os nomes e os tipos dos dados que serão armazenados.

Neste exemplo: as três variáveis são do tipo float porque podem guardar valores com parte decimal.

A declaração deve acontecer antes do uso das variáveis.

float nota1, nota2, media;

Comandos

Realizam as ações necessárias para resolver o problema.

Entrada: scanf lê as notas.

Processamento: a atribuição calcula a média.

Saída: printf apresenta o resultado.

scanf("%f", &nota1); media = (nota1 + nota2) / 2.0; printf("Media: %.2f\n", media);

Retorno

Devolve ao sistema um valor que indica como a função terminou.

Uso comum: return 0; indica que a função main terminou normalmente.

O retorno aparece antes da chave que fecha o bloco da função.

return 0;

Como as partes se conectam

Biblioteca

Prepara o acesso às funções que serão chamadas no código.

main

Abre o bloco principal e recebe as instruções executadas.

Variáveis

Definem onde os valores necessários ficarão armazenados.

Comandos

Executam leitura, cálculo, decisão, repetição ou saída.

Retorno

Finaliza a função principal e informa o término da execução.

Exemplos em C

Troque as abas para observar como a mesma estrutura básica permanece presente quando o programa passa de uma saída simples para problemas com entrada, cálculo e decisão.

Nível inicial

Mostrar uma mensagem

O programa apenas escreve uma frase na tela. Mesmo sendo curto, ele já apresenta biblioteca, função principal, comando e retorno.

Biblioteca

Libera o uso de printf.

main

Contém a instrução executada.

Comando e retorno

Mostra a mensagem e encerra a função.

mensagem.c
#include <stdio.h>

int main() {
    printf("Ola, mundo!\n");

    return 0;
}
Nível intermediário

Somar dois números inteiros

O programa recebe dois valores, calcula a soma e mostra o resultado. A estrutura passa a incluir declaração de variáveis e comandos de entrada.

Variáveis

a, b e soma guardam os dados.

Entrada

scanf recebe os dois números.

Processamento e saída

A atribuição soma os valores e printf mostra o resultado.

soma.c
#include <stdio.h>

int main() {
    int a, b, soma;

    scanf("%d", &a);
    scanf("%d", &b);

    soma = a + b;
    printf("Soma: %d\n", soma);

    return 0;
}
Nível intermediário

Calcular a média e decidir a situação

Além de entrada, cálculo e saída, o programa usa uma estrutura de decisão para escolher qual mensagem deve ser exibida.

Biblioteca e main

Mantêm a mesma função estrutural dos exemplos anteriores.

Variáveis e cálculo

As notas são armazenadas e usadas para obter a média.

Decisão

if e else selecionam a saída de acordo com o resultado.

situacao.c
#include <stdio.h>

int main() {
    float nota1, nota2, media;

    scanf("%f", &nota1);
    scanf("%f", &nota2);

    media = (nota1 + nota2) / 2.0;

    if (media >= 7.0) {
        printf("Aprovado\n");
    } else {
        printf("Reprovado\n");
    }

    return 0;
}

Erros comuns

Esquecer a biblioteca

Por que acontece: o aluno usa printf ou scanf sem incluir stdio.h.

Como evitar: identifique quais funções prontas serão utilizadas antes de escrever a main.

Escrever comandos fora da main

Por que acontece: as chaves são posicionadas sem atenção ao bloco da função.

Como evitar: confira se variáveis e comandos estão entre a chave de abertura e a chave de fechamento.

Usar variável antes de declarar

Por que acontece: o aluno começa pela leitura ou pelo cálculo e esquece de reservar os dados.

Como evitar: liste os valores necessários e declare cada variável antes do primeiro uso.

Perguntas rápidas

Qual parte indica onde a execução de um programa em C começa?

A função main. As instruções do programa são executadas a partir do bloco dessa função.

Por que o arquivo stdio.h aparece antes da função main?

Porque ele declara recursos de entrada e saída, como scanf e printf, que serão usados dentro do programa.

Em que ordem variáveis, comandos e retorno costumam aparecer dentro da main?

Primeiro são declaradas as variáveis, depois aparecem os comandos que resolvem o problema e, ao final, o retorno da função.
