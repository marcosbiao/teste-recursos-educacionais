Ideia central

O laço percorre os dados. O padrão define o que muda.

Muitos problemas diferentes compartilham a mesma estrutura: iniciar variáveis, repetir a leitura ou o acesso aos valores, atualizar o resultado parcial e apresentar a resposta final.

Estrutura que se repete

1. Inicializar

Prepare o estado

Use 0 para soma e contagem. Para maior e menor, prefira o primeiro valor válido.

2. Obter valor

Leia ou acesse

O valor atual pode vir do teclado, de um vetor ou de outra fonte.

3. Filtrar

Decida se participa

Uma condição pode selecionar apenas valores válidos ou de interesse.

4. Atualizar

Aplique o padrão

Some, conte ou compare o valor atual com o estado armazenado.

5. Finalizar

Produza a resposta

Calcule resultados derivados, como a média, e mostre os valores finais.

Exemplos progressivos

Somar cinco números

Nível básico

Leia cinco números inteiros, calcule a soma e informe quantos valores foram processados.

PadrõesSoma e contagem
Inicializaçãosoma = 0, quantidade = 0
Atualizaçãosoma += numero; quantidade++
soma_contagem.c
#include <stdio.h>

int main() {
    int i, numero;
    int soma = 0;
    int quantidade = 0;

    for (i = 0; i < 5; i++) {
        scanf("%d", &numero);
        soma += numero;
        quantidade++;
    }

    printf("Soma: %d\n", soma);
    printf("Quantidade: %d\n", quantidade);
    return 0;
}

Média das notas válidas

Nível intermediário

Leia seis notas. Considere apenas valores entre 0 e 10 e calcule a média das notas válidas.

Filtronota >= 0 && nota <= 10
PadrõesSoma, contagem e média
CuidadoDividir apenas se quantidade > 0
media_validas.c
#include <stdio.h>

int main() {
    int i, quantidade = 0;
    float nota, soma = 0.0, media;

    for (i = 0; i < 6; i++) {
        scanf("%f", &nota);

        if (nota >= 0 && nota <= 10) {
            soma += nota;
            quantidade++;
        }
    }

    if (quantidade > 0) {
        media = soma / quantidade;
        printf("Media: %.2f\n", media);
    } else {
        printf("Nenhuma nota valida.\n");
    }

    return 0;
}

Estatísticas até a sentinela zero

Nível intermediário

Leia inteiros positivos até que o usuário digite 0. Mostre soma, quantidade, média, maior e menor.

Paradanumero == 0
Inicialização de maior e menorPrimeiro valor válido
Padrões combinadosTodos os cinco
estatisticas.c
#include <stdio.h>

int main() {
    int numero, soma = 0;
    int quantidade = 0;
    int maior, menor;
    float media;

    scanf("%d", &numero);

    while (numero != 0) {
        if (quantidade == 0) {
            maior = numero;
            menor = numero;
        } else {
            if (numero > maior) maior = numero;
            if (numero < menor) menor = numero;
        }

        soma += numero;
        quantidade++;
        scanf("%d", &numero);
    }

    if (quantidade > 0) {
        media = (float) soma / quantidade;
        printf("Soma: %d | Quantidade: %d\n", soma, quantidade);
        printf("Media: %.2f | Maior: %d | Menor: %d\n", media, maior, menor);
    }

    return 0;
}

Erros comuns

Inicializar maior ou menor com zero

Por que acontece: o aluno repete a inicialização usada na soma.

Como evitar: use o primeiro valor válido como candidato inicial.

Dividir dentro do laço

Por que acontece: soma e quantidade ainda estão incompletas.

Como evitar: calcule a média depois que todos os valores forem processados.

Contar valores que não participam

Por que acontece: o contador é atualizado fora do filtro.

Como evitar: some e conte dentro da mesma condição de validade.

Perguntas rápidas

Por que a média normalmente é calculada depois do laço?

Porque ela depende da soma total e da quantidade final de valores considerados.

Qual é uma forma segura de inicializar maior e menor?

Usar o primeiro valor válido lido como candidato inicial para ambos.

Se apenas números positivos devem participar, onde soma e contagem devem ser atualizadas?

Dentro da condição que verifica se o valor é positivo, para que os dois estados considerem o mesmo conjunto de dados.

Fechamento pedagógico

Depois de estudar este infográfico, o aluno deve conseguir reconhecer qual padrão um problema de repetição exige, inicializar as variáveis adequadamente e atualizar soma, contagem, média, maior e menor de forma coerente.

Acumulador

Padrão de soma

×
Com 8, 3, 10 e 6: a soma começa em 0 e passa por 8, 11, 21 e 27.
Inicialização

Comece com 0, o elemento neutro da adição.

Atualização

O novo total usa o acumulado anterior e o valor atual.

Uso frequente

Total de vendas, pontos, idades ou valores lidos.

// antes do laço
soma = 0;

// dentro do laço
soma = soma + valor;
// forma equivalente
soma += valor;
Contador

Padrão de contagem

×
Com 8, 3, 10 e 6: a quantidade começa em 0 e termina em 4, pois quatro valores foram processados.
Inicialização

Use 0 para representar nenhuma ocorrência contada.

Atualização

Acrescente 1 quando ocorrer o evento que deve ser contado.

Posição da atualização

Dentro do filtro, quando apenas alguns valores participam.

// contar todos
quantidade++;

// contar apenas positivos
if (valor > 0) {
    quantidade++;
}
Resultado derivado

Padrão de média

×
Com 8, 3, 10 e 6: soma = 27, quantidade = 4 e média = 27 / 4 = 6,75.
Estados necessários

Uma soma acumulada e uma contagem coerente.

Momento do cálculo

Normalmente depois do laço, quando os dois valores estão completos.

Segurança

Verifique se a quantidade é maior que zero antes da divisão.

if (quantidade > 0) {
    media = (float) soma / quantidade;
}

// conversão evita divisão inteira
Seleção por comparação

Padrão de maior valor

×
Com 8, 3, 10 e 6: o candidato começa em 8, permanece 8, muda para 10 e termina em 10.
Inicialização segura

Use o primeiro valor válido, não um número arbitrário.

Comparação

Troque o candidato apenas quando aparecer um valor maior.

Estado mantido

Em cada instante, a variável guarda o maior visto até então.

maior = primeiroValor;

if (valor > maior) {
    maior = valor;
}
Seleção por comparação

Padrão de menor valor

×
Com 8, 3, 10 e 6: o candidato começa em 8, muda para 3 e permanece 3 até o final.
Inicialização segura

Use o primeiro valor válido para funcionar também com números negativos.

Comparação

Troque o candidato quando o valor atual for menor.

Estado mantido

A variável representa o menor valor já encontrado.

menor = primeiroValor;

if (valor < menor) {
    menor = valor;
}
