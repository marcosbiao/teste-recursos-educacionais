Ideia central

Um vetor de structs reúne vários registros do mesmo tipo

A struct define como um registro é organizado. O vetor reserva várias posições desse tipo. Assim, turma[0], turma[1] e turma[2] são alunos diferentes, mas todos possuem os mesmos campos.

Acesso completo: turma[i].media significa “campo média do aluno que está na posição i”.

A condição usa i < 3 porque o vetor possui três posições válidas: 0, 1 e 2.

1i = 0Acessa turma[0]: Lia, média 8.4.
2i = 1Acessa turma[1]: Caio, média 6.9.
3i = 2Acessa turma[2]: Nina, média 9.1.
4i = 3A condição fica falsa. O índice 3 não é acessado.

Nível básico

Exibir os registros da turma

alunos.c
#include <stdio.h>

struct Aluno {
    char nome[20];
    float media;
};

int main() {
    struct Aluno turma[3] = {
        {"Lia", 8.4},
        {"Caio", 6.9},
        {"Nina", 9.1}
    };

    for (int i = 0; i < 3; i++) {
        printf("%s: %.1f
",
               turma[i].nome,
               turma[i].media);
    }

    return 0;
}
Nível intermediário

Calcular o valor total do estoque

Cada produto contribui com preco × quantidade.

ProdutoCálculo
Caderno12,50 × 4 = 50,00
Caneta2,00 × 10 = 20,00
TotalR$ 70,00
estoque.c
#include <stdio.h>

struct Produto {
    char nome[20];
    float preco;
    int quantidade;
};

int main() {
    struct Produto produtos[2] = {
        {"Caderno", 12.50, 4},
        {"Caneta", 2.00, 10}
    };
    float total = 0;

    for (int i = 0; i < 2; i++) {
        total += produtos[i].preco
               * produtos[i].quantidade;
    }

    printf("Total: R$ %.2f
", total);
    return 0;
}
Nível intermediário

Buscar um livro pelo código

O laço percorre a coleção até localizar a chave procurada.

EtapaAção
PercorrerVisitar livros[i]
CompararTestar codigo
ResultadoMostrar título ou informar ausência
busca_livro.c
#include <stdio.h>

struct Livro {
    int codigo;
    char titulo[30];
};

int main() {
    struct Livro livros[3] = {
        {101, "Algoritmos"},
        {205, "Linguagem C"},
        {310, "Estruturas"}
    };
    int busca = 205;
    int posicao = -1;

    for (int i = 0; i < 3; i++) {
        if (livros[i].codigo == busca) {
            posicao = i;
            break;
        }
    }

    if (posicao != -1)
        printf("%s
", livros[posicao].titulo);
    else
        printf("Livro nao encontrado
");

    return 0;
}

Erros comuns

Trocar a ordem do índice e do campo

Por que acontece: o aluno tenta escrever algo como turma.media[i].

Como evitar: primeiro escolha o registro com [i], depois o campo com ponto: turma[i].media.

Ultrapassar o tamanho do vetor

Por que acontece: um vetor de tamanho 3 é percorrido até o índice 3.

Como evitar: use índices de 0 a 2 e uma condição como i < 3.

Usar toda a capacidade como quantidade real

Por que acontece: o vetor possui espaço para 100 registros, mas apenas parte foi preenchida.

Como evitar: mantenha uma variável quantidade com o número de registros válidos.

Perguntas rápidas

O que existe em cada posição de um vetor de structs?

Uma instância completa da estrutura, com todos os campos definidos no molde.

Como acessar a média do aluno que está na posição 2?

Use turma[2].media. O índice escolhe o aluno e o ponto escolhe o campo.

Um vetor com capacidade 50 sempre possui 50 registros válidos?

Não. Capacidade indica o espaço reservado. Uma variável de quantidade deve indicar quantas posições foram realmente preenchidas.

Molde do registro

Todos os elementos do vetor seguem a mesma estrutura.

A definição de struct Aluno determina quais campos cada posição da coleção terá. No exemplo, todo aluno possui nome e media.

struct Aluno { char nome[30]; float media; };
×

Vetor de structs

A coleção reserva várias posições do tipo composto.

A declaração struct Aluno turma[3]; cria espaço para três alunos. Cada posição guarda todos os campos de um aluno.

turma[0], turma[1] e turma[2] são três registros independentes.
×

Índice

O índice escolhe um registro da coleção.

Em turma[i], o valor de i determina qual aluno será acessado. Se i vale 1, a expressão seleciona o segundo registro.

Cuidado: em um vetor de tamanho 3, os índices válidos são 0, 1 e 2.
×

Campo

O ponto escolhe uma informação dentro do registro.

Depois de selecionar o aluno com turma[i], use o operador ponto para indicar o campo desejado.

  • turma[i].nome: nome do aluno atual
  • turma[i].media: média do aluno atual
×

Percurso

O laço faz o índice visitar todos os registros válidos.

O padrão for (i = 0; i < quantidade; i++) percorre apenas as posições preenchidas. Dentro do laço, turma[i] representa o registro atual.

printf("%s", turma[i].nome); mostra o nome de cada aluno durante o percurso.
