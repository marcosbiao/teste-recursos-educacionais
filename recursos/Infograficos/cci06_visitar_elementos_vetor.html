Ideia central

Percorrer é visitar uma posição por vez

O vetor guarda vários valores sob um único nome. Para acessar todos, o programa usa um índice que começa na primeira posição, avança uma unidade e para antes de alcançar o tamanho do vetor.

para i de 0 até tamanho - 1, usar vetor[i]
1

Comece em zero

Em C, a primeira posição de um vetor tem índice 0.

2

Respeite o tamanho

Se o tamanho é 5, os índices válidos vão de 0 a 4.

3

Acesse com o índice

Dentro do laço, vetor[i] representa o elemento visitado naquele momento.

Indica qual posição está sendo visitada naquele instante.

i = 0, 1, 2, 3, 4

Define quantas posições existem e onde o percurso deve parar.

Laço for

Controla o início, o teste e o avanço do índice.

for (i = 0; i < 5; i++)

Anatomia do percurso

1. Inicializar

Escolher a primeira posição

O índice começa em zero para apontar para o primeiro elemento.

i = 0
2. Testar

Verificar se a posição existe

O percurso continua enquanto o índice for menor que o tamanho.

i < tamanho
3. Acessar

Usar o elemento atual

O valor da posição atual é acessado pelo nome do vetor e pelo índice.

numeros[i]
4. Atualizar

Avançar para a próxima posição

Depois de usar o elemento, o índice aumenta uma unidade.

i++

Índice, posição e valor

Três ideias que não são iguais

Posição é a ordem usada na linguagem cotidiana. Índice é o número usado pelo programa. Valor é o conteúdo guardado.

A 1ª posição usa índice 0. A 5ª posição usa índice 4.

Exibir todos os elementos

Nível básico

O programa percorre o vetor e mostra o valor armazenado em cada posição.

exibir_vetor.c
#include <stdio.h>

int main() {
    int numeros[5] = {12, 7, 20, 5, 9};
    int i;

    for (i = 0; i < 5; i++) {
        printf("%d\n", numeros[i]);
    }

    return 0;
}

Encontrar o maior valor e sua posição

Nível intermediário

O índice permite saber não apenas qual é o maior valor, mas também em qual posição ele foi encontrado.

  • Inicialize o maior com o primeiro elemento.
  • Comece a comparação no índice 1.
  • Atualize o maior e a posição quando encontrar um valor superior.
ResultadoValor
Maior elemento20
Índice2
Posição humana3ª posição
maior_e_posicao.c
#include <stdio.h>

int main() {
    int numeros[5] = {12, 7, 20, 5, 9};
    int i;
    int maior = numeros[0];
    int indiceMaior = 0;

    for (i = 1; i < 5; i++) {
        if (numeros[i] > maior) {
            maior = numeros[i];
            indiceMaior = i;
        }
    }

    printf("Maior: %d, indice: %d\n",
           maior, indiceMaior);

    return 0;
}

Erros comuns

Usar i <= tamanho

Por que acontece: o aluno confunde o tamanho do vetor com o último índice válido.

Como evitar: use i < tamanho. Para tamanho 5, o último índice é 4.

Começar em i = 1

Por que acontece: a contagem cotidiana começa em um, mas os índices em C começam em zero.

Como evitar: em um percurso completo, inicie em i = 0.

Usar uma posição fixa

Por que acontece: o aluno escreve numeros[0] dentro do laço e repete sempre o primeiro valor.

Como evitar: use numeros[i] para que o acesso acompanhe o índice.

Perguntas rápidas

Quais índices são válidos em um vetor de tamanho 8?

Os índices válidos vão de 0 a 7.

Por que a condição costuma ser i < tamanho?

Porque o índice igual ao tamanho já estaria fora do vetor. A condição interrompe o laço antes desse acesso inválido.

O que vetor[i] representa dentro do laço?

Representa o valor armazenado na posição indicada pelo índice atual i.

Coleção indexada

O vetor

O vetor reúne vários valores do mesmo tipo sob um único nome. Cada valor ocupa uma posição identificada por um índice.

Mesmo exemplo: int numeros[5] = {12, 7, 20, 5, 9};
  • O nome da coleção é numeros.
  • O tamanho é 5.
  • Os índices válidos são 0, 1, 2, 3 e 4.
×
Localização

O índice

O índice funciona como um endereço dentro do vetor. Quando i muda, o acesso numeros[i] passa a apontar para outra posição.

Mesmo exemplo: quando i = 2, a expressão numeros[i] acessa o valor 20.
  • Índice não é o valor armazenado.
  • Índice 2 corresponde à terceira posição humana.
  • O índice deve permanecer dentro dos limites do vetor.
×
Limite do percurso

O tamanho

O tamanho informa quantos elementos o vetor possui. Como o primeiro índice é zero, o último índice válido sempre será tamanho - 1.

Mesmo exemplo: tamanho 5 significa índices de 0 até 4.
  • numeros[4] é válido.
  • numeros[5] está fora do vetor.
  • A condição segura é i < 5.
×
Controle da repetição

O laço for

O for reúne os três controles do percurso: valor inicial do índice, condição de continuidade e atualização.

Mesmo exemplo: for (i = 0; i < 5; i++)
  • i = 0 aponta para a primeira posição.
  • i < 5 impede acesso fora do vetor.
  • i++ avança para o próximo elemento.
