Problema a resolver

Registrar atendimentos até o operador informar a senha de encerramento

Em um posto de atendimento, o operador informa o número de cada senha atendida. Como não se sabe previamente quantas pessoas serão atendidas, a leitura deve continuar até que seja digitado o valor 0. A senha 0 é reservada para encerrar o registro. Ela controla a repetição, mas não representa um atendimento realizado.
EntradaLer a senha

Cada valor positivo representa uma pessoa atendida.

ControleComparar com 0

Enquanto a senha for diferente da sentinela, o laço continua.

EstadoAtualizar o contador

Somente senhas válidas aumentam o total de atendimentos.
