Problema a resolver

Uma corrida comunitária precisa cadastrar e classificar os participantes

A Corrida do Bairro registra nome e idade de cada participante. A idade define a faixa etária e o valor da inscrição. Ao final, o sistema deve informar o total arrecadado, a média de idade, a quantidade de participantes em cada faixa e um relatório individual das inscrições. Regras didáticas: Infantil, de 8 a 12 anos, paga R$ 20; Juvenil, de 13 a 17, paga R$ 25; Adulto, de 18 a 59, paga R$ 40; Sênior, a partir de 60, paga R$ 20. Idades abaixo de 8 não são aceitas nesta prova.

1#include <stdio.h>
2#define MAX 5
3
4int classificarFaixa(int idade) {
5 if (idade < 8) return -1;
6 if (idade <= 12) return 0;
7 if (idade <= 17) return 1;
8 if (idade <= 59) return 2;
9 return 3;
10}
11
12double calcularTaxa(int faixa) {
13 if (faixa == 1) return 25.0;
14 if (faixa == 2) return 40.0;
15 return 20.0;
16}
17
18Cadastroint main() {
19 char nomes[MAX][30];
20 int idades[MAX], faixas[MAX], contagem[4] = {0};
21 double taxas[MAX], total = 0, somaIdades = 0;
22
23 for (int i = 0; i < MAX; i++) {
24 scanf("%29s", nomes[i]);
25 do { scanf("%d", &idades[i]); } while (idades[i] < 8);
26 faixas[i] = classificarFaixa(idades[i]);
27 taxas[i] = calcularTaxa(faixas[i]);
28 contagem[faixas[i]]++;
29 total += taxas[i]; somaIdades += idades[i];
30 }
31
32Relatório for (int i = 0; i < MAX; i++)
33 printf("%s | %d anos | faixa %d | R$ %.2f\n", nomes[i], idades[i], faixas[i], taxas[i]);
34 printf("Arrecadacao: R$ %.2f | Media: %.1f\n", total, somaIdades / MAX);
35 return 0;
36}
