Problema a resolver

Calcular o M.M.C. entre dois números naturais.

Escreva um algoritmo que calcula o M.M.C. entre dois números naturais. Utilize uma função para calcular o M.M.C.. Os dois números naturais são fornecidos no escopo principal. Mostre o resultado em um procedimento.

Como encontrar o primeiro múltiplo de um número que também serve para o outro?

Este roteiro usa a ideia de tabuada: testar múltiplos do primeiro número até encontrar um valor divisível pelo segundo.

Pseudocódigo: função calcularMMC(numero1, numero2) multiploAtual recebe numero1 enquanto multiploAtual não for divisível por numero2 multiploAtual recebe multiploAtual + numero1 retornar multiploAtual procedimento mostrarResultado(mmc) mostrar mmc principal ler numero1 ler numero2 mmc recebe calcularMMC(numero1, numero2) mostrarResultado(mmc) #include <stdio.h> int calcularMMC(int numero1, int numero2) { int multiploAtual = numero1; while (multiploAtual % numero2 != 0) { multiploAtual = multiploAtual + numero1; } return multiploAtual; } void mostrarResultado(int mmc) { printf("M.M.C.: %d\n", mmc); } int main() { int numero1; int numero2; int mmc; printf("Digite o primeiro numero: "); scanf("%d", &numero1); printf("Digite o segundo numero: "); scanf("%d", &numero2); mmc = calcularMMC(numero1, numero2); mostrarResultado(mmc); return 0; }

Erros comuns tratados

Usar variável sem compreender o papel

Por que acontece: o aluno vê nomes como resultado ou candidato sem saber o que representam.

Como evitar: apresentar cada variável pelo papel antes do primeiro uso.

Parar o laço no momento errado

Por que acontece: o aluno confunde continuar enquanto o resto é diferente de zero com parar quando o resto é zero.

Como evitar: simular duas ou três tentativas antes de escrever o while.

Atualizar o múltiplo com o número errado

Por que acontece: o aluno soma numero2 ou soma os dois números, saindo da tabuada de numero1.

Como evitar: lembrar que multiploAtual deve continuar sendo múltiplo de numero1.
