O valor da variável muda: acompanhe o estado

Entenda como rastrear os valores de variáveis após atribuições e atualizações em programas simples em C.

Identificação pedagógica

Competência principal
CCI03

Manipular dados simples, variáveis, constantes e expressões preservando coerência do estado.

Competências secundárias
CCI02

Relacionar entrada, processamento e saída em um fluxo verificável.

Item trabalhado
Estado do programa

Acompanhar os valores atuais das variáveis durante a execução.

Tempo estimado
10 a 15 minutos

Uso rápido antes de analisar trechos de código com atribuições.

Estilo associado
Visual e sequencial

Rastreamento por linhas, tabelas e atualização gradual dos valores.

Pré requisitos mínimos

Variável e valor armazenado Comando de atribuição Expressões aritméticas simples Leitura linha por linha

Ideia central

Estado é o retrato dos valores atuais

O estado do programa mostra quais valores estão guardados nas variáveis em um determinado momento. Cada atribuição pode criar um valor inicial ou substituir o valor anterior.

x = 3 y = 2 estado atual
1

Observe a linha

Veja qual comando será executado naquele momento.

2

Use o valor atual

Resolva a expressão com os valores que as variáveis têm agora.

3

Atualize o estado

Substitua apenas o valor da variável que recebeu nova atribuição.

Como acompanhar o estado

Passo 1

Leia em ordem

Execute mentalmente o programa de cima para baixo. Em C, a ordem das linhas muda o resultado.

Passo 2

Crie uma tabela

Separe uma coluna para cada variável e registre o valor após cada comando relevante.

Passo 3

Atualize com cuidado

Quando houver atribuição, calcule o lado direito primeiro e depois grave o novo valor no lado esquerdo.

Passo 4

Verifique a saída

Quando aparecer um printf, use o estado atual para saber o que será mostrado.

Exemplo mínimo

Problema

Acompanhar os valores de x e y após cada atribuição.

Comando Estado após executar
int x = 3; x = 3
int y = 2; x = 3, y = 2
x = x + y; x = 5, y = 2
y = x - 1; x = 5, y = 4
printf Mostra 5 4

Código curto em C

#include <stdio.h>

int main() {
    int x = 3;
    int y = 2;

    x = x + y;
    y = x - 1;

    printf("%d %d", x, y);

    return 0;
}

Erros comuns

Achar que o valor antigo continua

Por que acontece: o aluno vê a variável como uma lista de valores, não como um espaço que guarda o valor atual.

Como evitar: após uma nova atribuição, risque mentalmente o valor antigo e registre o novo.

Ler atribuição como igualdade

Por que acontece: o símbolo = é confundido com igualdade matemática.

Como evitar: leia x = x + 1 como recebe o valor atual de x mais 1.

Atualizar fora de ordem

Por que acontece: o aluno calcula uma linha usando valores que ainda não existem naquele ponto do programa.

Como evitar: acompanhe as linhas na sequência e use sempre o estado atual.

Perguntas rápidas

Depois de uma atribuição, o que acontece com o valor anterior da variável?

Ele é substituído pelo novo valor. Para acompanhar corretamente, registre apenas o valor atual da variável.

Em x = x + 2, qual valor de x deve ser usado no lado direito?

Deve ser usado o valor atual de x, ou seja, o valor que estava guardado antes dessa linha ser executada.

Por que uma tabela ajuda a rastrear o estado do programa?

Porque ela mostra, linha por linha, como cada atribuição altera os valores armazenados nas variáveis.
