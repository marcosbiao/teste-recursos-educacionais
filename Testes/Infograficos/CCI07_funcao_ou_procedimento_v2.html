Ideia central

A escolha depende do papel do subprograma

Use função quando o subprograma precisa calcular, produzir e devolver um valor para outra parte do programa. Use procedimento quando ele deve apenas executar uma ação, como imprimir uma mensagem ou organizar uma saída.

1

Observe o verbo

Calcular, retornar e obter geralmente indicam função.

2

Procure a entrega

Se outra parte do programa precisa usar o resultado, o subprograma deve devolver esse valor.

3

Separe a ação

Se a tarefa é mostrar, ler ou informar algo, um procedimento pode ser suficiente.

Procedimento
void

Quando usar?

Quando não há valor a devolver.

Quando a tarefa é executar uma ação.

Quando o objetivo é imprimir, informar ou organizar uma etapa do programa.

Função em C

retorna valor

Tem tipo de retorno explícito, como int, float ou double. Normalmente usa return.

float calcularMedia(float a, float b) {
  return (a + b) / 2;
}

Procedimento em C

executa ação

Em C, costuma ser escrito como uma função void. Ele realiza uma ação e não devolve valor por return.

void mostrarMedia(float media) {
  printf("Media: %.2f", media);
}

Exemplo mínimo

Problema

Calcular a média de dois números e mostrar o valor obtido.

Parte do problema Estrutura adequada
Calcular a média Função, pois precisa devolver um valor float.
Mostrar o resultado Procedimento, pois apenas imprime uma informação.
Ler os números Escopo principal, para manter a entrada clara no main.

Código curto em C

#include <stdio.h>

float calcularMedia(float a, float b) {
    return (a + b) / 2;
}

void mostrarMedia(float media) {
    printf("Media: %.2f", media);
}

int main() {
    float n1, n2, media;

    scanf("%f", &n1);
    scanf("%f", &n2);

    media = calcularMedia(n1, n2);
    mostrarMedia(media);

    return 0;
}

Erros comuns

Usar procedimento para cálculo que precisa ser reutilizado

Por que acontece: o aluno imprime o resultado dentro do subprograma e não consegue usar o valor depois.

Como evitar: quando o resultado precisa voltar ao main, use função com return.

Usar return em procedimento void

Por que acontece: o aluno mistura a ideia de ação com a ideia de valor devolvido.

Como evitar: em void, execute a ação sem tentar devolver um valor.

Colocar tudo no main

Por que acontece: o aluno resolve a tarefa inteira em um único bloco de código.

Como evitar: separe cálculo e apresentação quando eles tiverem papéis diferentes.

Perguntas rápidas

Se o subprograma calcula uma média e o main precisa usar esse valor, qual estrutura é mais adequada?

Uma função, pois ela deve devolver a média calculada para o escopo principal.

Em C, como normalmente representamos um procedimento?

Com uma função do tipo void, usada para executar uma ação sem devolver valor.

Mostrar uma mensagem na tela geralmente combina mais com função ou procedimento?

Combina mais com procedimento, porque a tarefa principal é executar uma ação de saída.
