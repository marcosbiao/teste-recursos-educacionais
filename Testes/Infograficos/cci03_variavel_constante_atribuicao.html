Ideia central

Variável é um espaço de memória identificado por um nome

Em um programa, uma variável não é apenas um número. Ela é um local nomeado que guarda um valor. Quando uma atribuição acontece, o valor armazenado pode mudar e o estado do programa é atualizado.

1

Nome

A variável recebe um identificador para que o programa consiga acessar o valor.

2

Valor

O conteúdo armazenado pode ser consultado em expressões, cálculos e saídas.

3

Estado

Cada atribuição pode substituir o valor anterior e alterar o estado do programa.

Como pensar sobre memória com nome

Componente 1

Declaração

Indica o tipo e o nome da variável. Exemplo: int pontos;

Componente 2

Atribuição

Guarda um valor em um nome. Exemplo: pontos = 5;

Componente 3

Constante

Guarda um valor que não deve mudar durante a execução. Exemplo: const int BONUS = 10;

Componente 4

Atualização

Usa o valor atual para gerar um novo valor. Exemplo: pontos = pontos + BONUS;

Mudança de estado em uma variável

O nome permanece o mesmo, mas o valor armazenado pode ser substituído por uma nova atribuição.

Exemplo mínimo

Problema

Registrar uma pontuação inicial, adicionar um bônus fixo e mostrar a pontuação final.

Parte Aplicação no exemplo
Variável pontos guarda a pontuação atual.
Constante BONUS representa um valor fixo.
Atribuição pontos = pontos + BONUS; atualiza o valor armazenado.
Estado Antes: 5. Depois: 15.

Código curto em C

#include <stdio.h>

int main() {
    int pontos;
    const int BONUS = 10;

    pontos = 5;
    pontos = pontos + BONUS;

    printf("Pontuacao final: %d", pontos);

    return 0;
}

Erros comuns

Confundir nome com valor

Por que acontece: o aluno lê o identificador como se ele fosse o próprio número.

Como evitar: pense no nome como uma etiqueta para acessar o valor guardado na memória.

Usar variável sem valor definido

Por que acontece: a variável é declarada, mas recebe uso antes de uma leitura ou atribuição.

Como evitar: garanta que todo valor usado no cálculo já tenha sido inicializado.

Achar que constante pode mudar

Por que acontece: o aluno trata constante como uma variável comum.

Como evitar: use constante apenas para valores fixos, como taxas, limites e fatores definidos.

Perguntas rápidas

O que muda quando executamos pontos = pontos + BONUS?

O valor atual de pontos é consultado, somado ao valor fixo de BONUS e depois armazenado novamente em pontos.

Qual é a diferença entre variável e constante?

A variável pode receber novos valores durante a execução. A constante representa um valor fixo que não deve ser alterado.

Por que uma variável precisa ter nome?

Porque o nome permite localizar, consultar e atualizar o valor armazenado durante a execução do programa.
